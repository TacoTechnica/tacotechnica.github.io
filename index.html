Rhythm Game Online Songs + Leaderboards Mod

I made a mod and server that lets people play, upload and download custom beatmaps for the rhythm game UNBEATABLE [white label]
This includes a user registration system, a high score leaderboard, and an uploading system on discord with a simple approval/rating system to encourage feedback.
The entire system has been running since April 2022. It requires little maintenence on my behalf to function, creators upload and verify their maps using the system.
There are currently over 130 beatmaps uploaded.
This mod has formed a community of over 70 members who make and share custom maps for the game. I am incredibly happy to see my work help so many fans of the game!

Video Engagement Chrome Extension

I worked with a team of 4 to create Engage Me: A chrome extension that tracks user engagement and watch time on YouTube videos and lecture recordings (this was before YouTube implemented this feature themselves!)
I designed the front end facing component that would track which intervals the user watched (AKA didn't skip), combine them together, and send them to the server
Our project won 4th place at the MHacks 2019 hackathon

Fully Autonomous Minecraft Speedrun

I programmed the first bot ever to successfully beat the game Minecraft on its own in a Random Seed world
With the help of a pathfinding library, the bot dynamically collects resources and accomplishes increasingly complicated tasks by reacting to the state of the world
The repository currently sits at over 300 stars on github
The community around this mod has over 750 members and the project has been featured in a video with 10 million views
On May 25th, 2021 the bot was streamed beating Minecraft, becoming the first fully autonomous random seed speedrun in the game's history
